EN: In order for the Python script to function properly, it is essential to have the "image.png" file stored within the same directory as the .py or .exe file. Feel free to replace "image.png" with any other PNG image of size 256x256. You can also modify line 44-45 (labels 4 and 5) to match the selected image. The captured credentials are securely stored in a "log.txt" file located in the same directory as the aforementioned files. If the log file does not exist, it will be automatically created upon initial execution. The text file can concurrently store all "stolen" credentials. However, it is worth mentioning that even if the victim declines the prompt and clicks the "No" button, the captured credentials will still be stored anyway.

IT: Per far funzionare lo script Python, è necessario che l'immagine.png si trovi nella stessa directory del file .py o .exe.
"image.png" può essere liberamente sostituito con un'altra immagine png di dimensioni 256x256, è possibile anche modificare le righe 44-45 labels 4 e 5) per corrispondere all'immagine scelta. Le credenziali vengono salvate in un file chiamato log.txt, anch'esso situato nella stessa directory degli altri due file (la prima volta il file viene creato automaticamente). Il file di testo può memorizzare contemporaneamente tutte le "credenziali rubate". Anche se la vittima preme il pulsante "No", le credenziali vengono comunque memorizzate.
